Browser automation agents powered by large language models (LLMs) have shown promise in executing web-based tasks, but they face persistent challenges: brittle execution when web pages change, high token consumption from repeated trial-and-error, and the inability to retain and reuse successful strategies across sessions. We present OpenSkynet, an open-source framework that addresses these limitations through three integrated mechanisms: (1) a \textbf{self-healing skill executor} that automatically diagnoses and repairs failed automation steps by analyzing screenshots and DOM state via multimodal LLM reasoning; (2) a \textbf{skill learning pipeline} that extracts, refines, and merges reusable automation skills from observed trajectories; and (3) a \textbf{three-tier memory system} that progressively promotes procedural knowledge from working memory to persistent long-term storage, enabling cross-session skill improvement. OpenSkynet also introduces a multi-agent subagent architecture with permission-based tool isolation, parallel execution, and configurable browser backends. To evaluate self-healing and skill generalization under adversarial conditions, we propose the \emph{Evil Skill Generalization Benchmark}, comprising 60 tasks across 20 templates with deliberately obfuscated web interfaces (hidden elements, injected content, multi-layer consent dialogs).
